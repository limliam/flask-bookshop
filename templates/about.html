About Flask-Bookshop

Flask-Bookshop is a web application built with Flask that allows users to browse, add, edit, and delete books from a bookstore. It consumes a RESTful API for book data.

Setup Instructions

  1. Clone the repository from GitHub.
  2. Create a virtual environment and activate it.
  3. Install the required dependencies using `pip install -r requirements.txt`.
  4. Set the API_URL environment variable to the URL of your API.
  5. Run the Flask app using `python app.py`.

Connecting to the API

To connect Flask-Bookshop to the API, set the API_URL environment variable to the URL of your API. For example: export BOOKSHOP_API_URL='http://your-api-url.com'
